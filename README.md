## Quick start to use local
___
1. Fill the `user.properties` of credential data about user:
```
   demoqa.userName=
   demoqa.password=
 ```
4. Start all tests by terminal
   `./gradlew test`
   or only one test 
   `./gradlew test --tests BookStoreTests.noLogTest`
